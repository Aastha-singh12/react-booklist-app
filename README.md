# Book List App
The Book List App is a simple, user-friendly React application that displays a list of books. Users can search for books by title or author and view additional details by toggling a brief description.

# Features
✔️ View Books – Displays each book's title, author, and publication year.
✔️ Toggle Description – Allows users to show/hide a brief book summary.
✔️ Search Functionality – Easily search books by title or author.
✔️ State Management – Uses React’s useState hook for handling book data and search queries.
✔️ Responsive Design – Optimized for different screen sizes.

🛠 Technologies Used
React.js – Frontend framework
JavaScript (ES6) – Logic and interactivity
CSS Modules / Inline Styles – Styling
